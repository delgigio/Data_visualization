Coffee export for the highest producing countries worldwide until 2013

For this graph we wanted to map out the highest producing countries, for this we first checked the countries with the overall sum of production to be the greatest and took the first six as indication, from number seven to the last one was too low to be considered. There was no need to reformat the data or do preprocessing of the data. Set the first six countries names is a python list and used plotly.express library to do the graph and then set it to html to be visualised on our website.
